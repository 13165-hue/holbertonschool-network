# Holberton School - Network Basics

This repository contains solutions for the Holberton School Network Basics project.

## Project Structure

- `basics_0/` - Basic networking concepts
  - `0-OSI_model` - OSI model questions and answers

## Topics Covered

- OSI Model (layers and organization)
- Types of Networks (LAN, WAN)
- MAC and IP addresses
- TCP/UDP protocols
- Network ports
- ICMP and ping

## Requirements

- Bash scripts must be executable
- All files must end with a newline
- All scripts must pass shellcheck
- First line: `#!/usr/bin/env bash`
- Second line: comment explaining the script
